# Bandage E-Commerce

Bandage is a modern, responsive e-commerce frontend application built with React and Tailwind CSS. It features a clean design with multi-page navigation, product listings, detailed product views, and a dedicated team/about information architecture.

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Modern UI/UX**: Built with Tailwind CSS for a sleek, contemporary look.
- **Product Management**: Browse through various categories and view detailed product information.
- **Interactive Components**: Smooth transitions, mobile-friendly navigation menus, and interactive galleries.
- **Team and About Sections**: Comprehensive pages to showcase community and corporate identity.

## Tech Stack

- **React**: Modern functional components with hooks.
- **React Router**: Seamless client-side navigation.
- **Tailwind CSS**: Utility-first CSS framework for fast styling.
- **Lucide React**: High-quality vector icons.
- **Vite**: Rapid development and build tool.

## Project Structure

- `src/pages`: Individual page components (Home, Shop, Product, About, Team, Contact).
- `src/components`: Reusable UI elements (Product Cards, Navbar, Footer).
- `src/layout`: Main layout wrappers and global navigation.

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
